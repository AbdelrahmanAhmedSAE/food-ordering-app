# Food Ordering App

> This is a **Food Ordering Web Application** designed for full-stack portfolio purposes.
> The project is structured using Agile methodology with incremental Sprints.

---

## Project Overview

- **Frontend:** Customer & Admin pages
- **Backend:** REST API with PostgreSQL + PostGIS
- **Features:** Menu browsing, Cart, Orders, Admin CRUD, Geospatial delivery validation
- **Design:** Clean architecture, optional enhancements for portfolio (images, notes, audit logs)
